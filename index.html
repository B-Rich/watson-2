watson

a ruby gem

watson is a simple command line tool for generating a wiki. you pass it a directory with your notes. these files are run through markdown, threaded into a layout you provide, and ouputed into a 'wiki' directory in the notes directory. A web server is fired up to serve them to your locally. top hat also creates an index page with a list of all other pages. of course, syntax is highlighted.

install

      >> [sudo] gem install watson

setup

      >> mkdir ~/notes ~/notes/layouts
      >> touch ~/notes/layouts/layout.html

generate

      >> watson ~/notes

contribute

watson's code rests in github. if you find inspiration, please take action! patch a bug, add a feature. If you don't know ruby, or even if you do, the best way to contribute is to send me an email with your feedback. the moral sends me into a coding frenzy.
